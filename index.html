Efek Suara Notifikasi Synth

Aktifkan nada sinyal web audio yang menenangkan saat menyelesaikan tugas/pomodoro.

Ekspor / Impor Data Akademik

Cadangkan database local-first Anda dalam bentuk file JSON terenkripsi untuk dipindahkan ke peranti lain.

Prinsip Keamanan & Portabilitas Offline-First

  • Data akademik disimpan secara penuh di dalam browser lokal Anda (LocalStorage) untuk performa instan tanpa server.
  • Tidak ada API Key atau kredensial sensitif yang tersimpan keras (Hardcoded) dalam kode demi kepatuhan keamanan.
  • Lakukan pencadangan (backup) berkala untuk mengamankan data perkuliahan jangka panjang Anda.

Zona Berbahaya: Reset Database

Hapus seluruh data mata kuliah, agenda tugas, riwayat pomodoro, dan data skripsi secara permanen dari peranti ini.
